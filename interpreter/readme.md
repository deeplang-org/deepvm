### 主要结构体

---

```
DEEPStack(操作数栈)
DEEPInterpFrame(函数帧)
DEEPExecEnv(运行环境)
DEEPModule(从loader得到的module)
```

### 主要函数

---

```
exec_instructions(执行代码块内指令)
call_main(为main函数创建帧，执行main函数)
```
